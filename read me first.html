Nani Fighter

How To Start For Novice

The .py files in this directory are the two versions of the game Nani Fighter.

For running these .py programs requires two programs to be installed in your computer:

1.Python 3.3

2.Pygame for python 3.3

If you don't have one of the above you must download and install it.

If you don't have no.1 click on the link no.1. Got the website? Now download the installer of Python 3.3.5 according to your operating system.If you are using windows download the Windows x86 MSI installer from the list of files.

Installed? Now download and install the no.2 that is Pygame for python 3.3 from the above link no.2. according to your operating system. Likewise, if you are still a windows user download pygame-1.9.2a0.win32-py3.3.msi from the list of windows installer. Installed? You are done.

Congrats

You can now open the .py files and play the two different versions of Nani Fighter.
